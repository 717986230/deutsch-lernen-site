# 更新日志

面向用户的变化记录，最新在最上。

## 2026-07-23

### ✨ 新增
- **账号安全设置页**：个人主页新增「⚙️ 账号设置」——可修改密码、退出其它所有设备、注销账号。
- **注销账号**：带不可恢复警示 + 二次确认（密码账号验密码、第三方账号输用户名确认），永久删除云端账号与学习数据（本地记录不受影响）。
- **退出登录**：现在会真正吊销服务器端会话（此前仅清本地）。- **搜索体验**：词句大全搜索结果高亮命中关键词；记住最近 8 条搜索，一键重搜。

## 2026-07-22

### ✨ 新增
- **版块直达链接**：地址栏加 `#spell`、`#quiz` 等即可直接打开对应版块，方便收藏和分享给同学。
- **测验错题进错词本**：词汇类测验结算页会列出本次错题（德语 + 中文 + 🔊发音），一键「拼写巩固错词」即可把错词写入拼写错词本，随后针对性重练。
- **今日目标可调**：每日学习目标可在 10 / 20 / 50 / 100 之间切换，当天首次达标会有提示祝贺。
- **首页「继续上次学习」**：一键回到上次的词句级别分类或拼写场景，不用再层层点回去。
- **账号安全能力**：支持注销账号（删除全部账号数据，不可恢复）、修改密码（改后自动踢掉其他设备）、退出当前 / 全部设备。本次为后端接口上线，设置页里的操作入口将在下个版本提供。

### 🔧 修复
- **微信里按返回键不再退出网站**：返回键现在会回到上一个版块，而不是直接离开站点。
- **英语模式更一致**：底部导航不再显示（德语专属的）「短文」；测验级别标签改为 入门 / 中考 / 高考 / 四级 / 六级。
- **弱网下「我的」页不再误报**：网络不好时不再错误提示「请重新登录」，改为显示离线资料卡并提供重试按钮。

### ⚡ 优化
- **注册 / 登录防滥用保护**：对高频注册和反复输错密码增加频次限制，保护你的账号不被暴力尝试。
